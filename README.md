# ITM8 Attendance System

The ITM8 Attendance System simplifies the process of tracking student attendance, managing schedules, and generating reports. It features a robust Python/FastAPI backend and a responsive React frontend.

## Quick Links

- [Setup Guide](docs/setup.md): Instructions on how to install and run the project.
- [Technology Stack](docs/technologies.md): Detailed information about the libraries and frameworks used.

## Key Features

- **User Management**: Role-based access control (RBAC) for admins, teachers, and students.
- **Attendance Tracking**: Real-time checking in and out functionality.
- **Reporting**: Automated attendance reports and performance analytics.
- **Modern UI**: Clean, responsive interface built with Tailwind CSS.
